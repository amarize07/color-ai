# Color AI - PaletteNet TFLite

تطبيق ويب لتوليد لوحات ألوان متناسقة باستخدام نموذج TensorFlow Lite.

## المميزات

- 🎨 توليد 3 ألوان متناسقة من أي لون مختار
- ⚡ نموذج TFLite يعمل بالكامل داخل المتصفح
- 🔒 آمن وسريع - بدون إرسال البيانات للخادم
- 📱 واجهة ذات استجابة عالية (Responsive)
- 🌐 دعم اللغة العربية

## الملفات

- `index.html` - صفحة الويب الرئيسية
- `palette_model_quantized.tflite` - نموذج التنبؤ بالألوان
- `.nojekyll` - تعطيل معالج Jekyll في GitHub Pages
- `.gitattributes` - معالجة صحيحة للملفات الثنائية

## الاستخدام

1. اختر لوناً من لاقط الألوان
2. انقر على زر "✨ توليد اللوحة"
3. شاهد الألوان الثلاثة المقترحة
